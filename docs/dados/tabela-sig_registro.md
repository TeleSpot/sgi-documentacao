# Tabela sig_registro

Representa a **linha principal da planilha**.

## Chave

- `id` (UUID)

## Campos principais

- cliente
- ordem_servico
- serial

## Função

- ponto central de vínculo entre todas as etapas
- base para rastreabilidade e relatórios

Cada registro corresponde a **um cliente/projeto** no fluxo operacional.
