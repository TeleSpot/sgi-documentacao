# Fórmulas e Derivações

Nesta fase, o sistema **replica a planilha**, sem recalcular automaticamente.

Futuras fórmulas incluem:

- margem bruta
- margem de contribuição
- desvio de custo
- prazo médio por etapa
- indicadores por instalador, franqueado ou UF

As fórmulas serão implementadas:

- em Python (service layer)
- ou via views SQL
