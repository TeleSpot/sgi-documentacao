# Status e Marcos

## Status

O status atual do projeto é armazenado em `sig_andamento`.

Inicialmente:

- campo livre (texto)
- espelha o comportamento da planilha

Futuramente:

- enumeração controlada
- regras automáticas de transição

## Marcos

Marcos são definidos por datas:

- previsto
- realizado

A presença de uma data realizada indica conclusão.
